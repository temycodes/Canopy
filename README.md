## Canopy

A lightweight team management web app for organizing members, tracking availability, and maintaining a clear team structure.

## Features

* Create, edit, and delete users
* Track availability: **Online**, **Offline**, **Away**
* Filter users by status for quick visibility

## Tech Stack

* TypeScript
* React
* Zustand
* Bootstrap

## Purpose

Built for simple, fast team coordination with a clean and minimal interface.
